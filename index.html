About Me

Hello! I'm Scott Faust, and I'm on a journey to harness the transformative power of technology as a software engineer. My passion lies in not just understanding the intricacies of code but in using it as a tool to make a real-world impact.

Born and raised in the nurturing environment of Denver, Colorado, my fascination with technology started at a young age. I found myself intrigued by the endless possibilities that software engineering presented – it was like having a language to converse with the future. Joining the Airforce right out of high school, I served as an Aircraft Mechanic, where I got my first hands-on experience with sophisticated technology. This experience wasn't just about understanding machinery; it was a lesson in precision, discipline, and the importance of every small part in a larger system.

After my retirement in 2018, I embarked on a new mission – to become a software engineer. My journey at the University of Nebraska at Omaha isn't just an academic pursuit; it's a path to realizing my dream of making the world a better place. Here, I'm not only honing my skills in coding and software development but also learning how to leverage these skills to create accessible, user-friendly solutions.

My vision is a world where technology is not just an elite privilege but a tool accessible to everyone, everywhere. I believe that in the right hands, technology can be a great equalizer, breaking down barriers and opening doors to limitless possibilities. Whether it's developing applications that improve everyday life, creating systems that make information more accessible, or contributing to groundbreaking innovations, I'm committed to using my skills for the greater good.

As I stride forward in my career, I carry with me a commitment to excellence, a passion for learning, and a steadfast belief that technology, when used wisely and ethically, can indeed make the world a better place for all.

Projects

Rock Paper Scissors

The "Rock Paper Scissors" project is a web-based implementation of the classic game. It includes a user interface for selecting rock, paper, or scissors, and a scoring system. The JavaScript file (web_file.js) contains functions for handling the game logic, including generating computer choices, determining wins, losses, and draws, and updating the score display. This project demonstrates understanding of basic game mechanics, DOM manipulation, and event handling in JavaScript, as well as the ability to create interactive web applications.

Sudo Battleship

"Sudo Battleship" is a web-based implementation of the classic Battleship game. It features a 5x5 grid where players can click on cells to make guesses. The game randomly assigns a ship location, and players aim to find it. The grid dynamically responds to user interactions, changing colors to indicate hits or misses. A reset button allows restarting the game. This project showcases abilities in DOM manipulation, event handling, and logic implementation in JavaScript, along with responsive web design using CSS.

View Project

Ball Pit

The "Ball Pit" project is an interactive web application featuring a canvas-based animation of bouncing balls. It includes a JavaScript file that handles the creation and animation of balls on the canvas. Users can interact with the application to add balls, change their speed, size, and color. The project demonstrates proficiency in canvas manipulation, object-oriented programming, and interactive web design. It's a good example of dynamic visual effects and user interaction in web applications.

View Project

Space Invaders

The "Space Invaders" project is a complex JavaScript-based game involving multiple classes for different game elements like enemies, bullets, and the player's ship. The GameClass.js file orchestrates the overall game logic, managing game states, user inputs, and screen updates. It also handles collision detection, score tracking, and dynamic difficulty adjustment. This project demonstrates advanced skills in object-oriented programming, game development concepts, event handling, and managing complex interactions between various game components.

View Project

Matrix Style Background

The "Matrix Style Background" project is an HTML page featuring a canvas element, where a Matrix-inspired rain animation is created using JavaScript. It showcases JavaScript skills in creating dynamic, real-time animations and handling canvas API for rendering. The animation uses Japanese characters, falling vertically at different speeds to mimic the iconic digital rain effect from the Matrix movies. This project demonstrates proficiency in graphical effects, animation, and creative use of web technologies for immersive visual experiences.
